# Scalable Categorization of Satellite Images

A Django web app that uses deep learning to automatically categorize satellite images.  
Supports dataset upload, model training, and real-time classification with optimized accuracy.

**Tech Stack:** Python, Django, TensorFlow, HTML, CSS, Bootstrap, SQLite3
